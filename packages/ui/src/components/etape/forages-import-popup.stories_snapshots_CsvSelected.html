Modification des forages

Type de fichier

Vous pouvez déposer un fichier de forages pour modifier ou supprimer les propriétés des forages. Ils doivent avoir les champs suivants : "nom" (nous vous conseillons de ne pas dépasser 3 caractères), "description", "type" (rejet, captage ou piézomètre), "profondeur" (en NGF), "x", "y". Seul le champ description peut être vide.
